# Frontend Source Layout

Route files live in `src/app/`.

Domain behavior should live in:

- `src/features/auth/`
- `src/features/projects/`
- `src/features/tasks/`

Shared concerns live in:

- `src/components/shared/`
- `src/components/ui/`
- `src/providers/`
- `src/services/http/`
- `src/contracts/`
